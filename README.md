# 🔐 SecurePass Pro

A modern and secure password generator built using HTML, CSS, and JavaScript that helps users create strong passwords instantly to improve online security.

---

## 📖 Overview

SecurePass Pro is a lightweight web application designed to generate secure and random passwords based on user preferences. Users can customize password length and choose whether to include uppercase letters, lowercase letters, numbers, and special symbols.

The application also evaluates password strength and provides a one-click copy feature for convenience.

---

## ✨ Features

* 🔑 Generate random and secure passwords
* 📏 Custom password length selection
* 🔠 Include uppercase letters
* 🔡 Include lowercase letters
* 🔢 Include numbers
* ✳️ Include special symbols
* 📊 Password strength indicator
* 📋 One-click copy to clipboard
* 🎨 Clean and responsive user interface
* ⚡ Fast client-side processing

## 🚀 How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/yourusername/SecurePass-Pro.git
```

2. Open the project folder.

3. Launch `index.html` in any modern web browser.

4. Start generating secure passwords.

---

## ⚙️ How It Works

1. Enter the desired password length.
2. Select the character types:

   * Uppercase Letters
   * Lowercase Letters
   * Numbers
   * Symbols
3. Click **Generate Password**.
4. The system creates a random password.
5. View the password strength.
6. Click **Copy Password** to copy it to the clipboard.
